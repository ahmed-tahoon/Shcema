User table (users):

Column Data Type Constraints Description
user_id INT (PK) Auto-increment, NOT NULL Primary key, unique identifier for users.
user_status ENUM NOT NULL, DEFAULT 'Active' User status (InActive, Active, Paused, Deleted).
user_type ENUM NOT NULL, DEFAULT 'Normal_User' User type (Admin, Moderator, Trader_User, Normal_User).
user_nick_name VARCHAR(255) Nickname or alias of the user.
user_name VARCHAR(255) NOT NULL Full name of the user.
user_email VARCHAR(255) NOT NULL, UNIQUE Email address of the user.
user_password VARCHAR(255) NOT NULL Encrypted password of the user.
user_telephone VARCHAR(20) Telephone number of the user.
user_address VARCHAR(255) Address of the user.
user_created_at DATETIME NOT NULL, DEFAULT CURRENT_TIMESTAMP Timestamp of user registration.
  1. User log table (user_log):
Column Data Type Constraints Description
userlog_id INT (PK) Auto-increment, NOT NULL Primary key, unique identifier for logs.
user_id INT (FK) NOT NULL Foreign key linking the log entry to the user.
userlog_ip VARCHAR(45) NOT NULL IP address of the user during the log.
userlog_timestamp DATETIME NOT NULL, DEFAULT CURRENT_TIMESTAMP Timestamp of the log entry.
  1. User alerts table (user_alerts):
Column Data Type Constraints Description
useralert_id INT (PK) Auto-increment, NOT NULL Primary key, unique identifier for alerts.
user_id INT (FK) NOT NULL Foreign key linking the alert to the corresponding user.
useralert_status ENUM NOT NULL Alert status (Unseen, Saw).
useralert_type ENUM NOT NULL Alert type (Primary, Secondary, Success, Danger, Warning, Info, Light, Dark).
useralert_first_seen DATETIME NOT NULL, DEFAULT CURRENT_TIMESTAMP Timestamp for the first time the user saw the alert.
useralert_last_seen DATETIME NOT NULL, DEFAULT CURRENT_TIMESTAMP Timestamp for the last time the user saw the alert.

Admarket table (admarket):

Column Data Type Constraints Description
adm_id INT (Primary Key) Auto-increment Unique identifier for admarket record.
adm_user_id INT (Foreign Key) NOT NULL References the user who posted the market ad.
adm_admin_id INT (Foreign Key) References the admin who accepted the market ad.
adm_cat_id INT (Foreign Key) NOT NULL References the category of the market ad.
adm_subcat_id INT (Foreign Key) NOT NULL References the subcategory of the market ad.
adm_status ENUM NOT NULL, DEFAULT 'Active' Status of the market ad (InActive, Active, Paused, Deleted).
adm_type ENUM NOT NULL Type of the market ad (New, Used).
adm_title VARCHAR(255) NOT NULL Title of the market ad.
adm_description TEXT NOT NULL Description of the market ad.
adm_price DECIMAL(10, 2) Price of the market item or service.
adm_location VARCHAR(255) Location of the market item or service.
adm_contact ENUM NOT NULL Contact method for the market ad (Phone, Email, Both).
adm_pictures JSON JSON array of picture URLs for the market ad.
adm_viewscounter INT NOT NULL, DEFAULT 0 Counter for the number of views the market ad received.
adm_TIMESTAMP TIMESTAMP NOT NULL, DEFAULT CURRENT_TIMESTAMP Timestamp of the market ad posting.
